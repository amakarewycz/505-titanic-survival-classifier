Titanic Survivability

Overview

Improve existing model for predicting survivability on the titanic disaster

Baseline Model

A Logistic Regression Model had the best performance over naive bayes, k-nearest neighbors and random forest

The AUC is 77.3

Additional EDA

From the feature importance chart, it seems features like being a youthful female is a strong predictor at surviving the Titanic disaster. Added a synthetic feature, "young female", F(young female) = F(female)==true and (F(Age (20, 28]) == 1 or F(Age (28, 38]) == 1) to capture this and re-trained several models to see if it made any difference in the prediction outcomes.

Also looked at a couple more synthetic variants for "young female". Looked at "young male", "older male", "young female" in "Class 3".

Only "young female" had improvements to the model

Model Selection

These are the results

Logistic Regression shows 2 point increase across all scores vs. the baseline.

Naive bayes is unchanged.

Random forest is a little worse off than before, f1 score suffers the most reduction by 1.3 points

Chosen Model

logistic regression

Results
